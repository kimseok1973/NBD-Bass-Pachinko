# NBD-Bass 統合モデル: パチンコ参加者動態の閾値効果補正

Bass 拡張モデル（3状態 ODE）に **NBD（Negative Binomial Distribution）** を組み合わせ、
「年1回以上プレイ」という閾値観測による歪みを補正したベイズ推定フレームワーク。
さらに $p_1(t)$ の関数形（Gaussian vs 規制ダミー）を比較検討し、識別可能性の限界を明らかにした研究プロジェクト。

---

## 1. 動機

従来の `Pachinko_3State_P1tP2t` では「パチンコ参加人口」の年次推移を直接観測として用いたが、$p_1$ と $p_2$ の識別性が不十分だった。

**仮説**: 「年1回以上」の閾値は、**真の離脱**と**低頻度化による閾値下落（NBD 閾値効果）**を区別できない。見かけの $p_2$ は両者の合成である。

---

## 2. 数理モデル全体像

### 2.1 個人レベルの遊戯頻度モデル（NBD）

各個人 $i$ の年間遊戯回数 $X_i$ は Poisson、強度 $\lambda_i$ は Gamma 分布:

$$X_i \mid \lambda_i \sim \text{Poisson}(\lambda_i T), \qquad \lambda_i \sim \text{Gamma}(r(t), \alpha(t))$$

ここで $T = 1$ 年、$r$ は形状、$\alpha$ は rate（逆スケール）。$\lambda$ を周辺化した分布は **Negative Binomial Distribution (NBD)**:

$$P(X_i = k \mid r, \alpha) = \frac{\Gamma(r + k)}{\Gamma(r) \, k!} \left(\frac{\alpha}{\alpha + 1}\right)^r \left(\frac{1}{\alpha + 1}\right)^k$$

集団平均: $\mathbb{E}[\lambda] = r / \alpha$、分散 $\mathbb{V}[\lambda] = r / \alpha^2$。

### 2.2 ゼロ確率と閾値補正

「年1回以上プレイ = アクティブ」の確率:

$$P(X \geq 1 \mid r, \alpha) = 1 - \left(\frac{\alpha}{\alpha + 1}\right)^r$$

### 2.3 Bass 拡張 3状態 ODE

正規化状態 $s + a + x = 1$（$s$: 潜在、$a$: アクティブ、$x$: 離脱）:

$$\frac{ds}{dt} = -\bigl(p_1(t) + q_1 a\bigr) s$$

$$\frac{da}{dt} = \bigl(p_1(t) + q_1 a\bigr) s + \alpha \bigl(p_1(t) + q_1 a\bigr) x - p_2(t) \, a$$

$$\frac{dx}{dt} = p_2(t) \, a - \alpha \bigl(p_1(t) + q_1 a\bigr) x$$

$q_2 = 0$ 固定。$p_2(t)$ は常にガウス摂動:

$$p_2(t) = p_{2,\text{base}} + A_2 \exp\!\left(-\frac{1}{2}\left(\frac{t - c_2}{w_2}\right)^2\right)$$

### 2.4 $p_1(t)$ の関数形（本研究で比較した3種）

**(A) ガウス版** — 4パラメータ:

$$p_1(t) = p_{1,\text{base}} + A_1 \exp\!\left(-\frac{1}{2}\left(\frac{t - c_1}{w_1}\right)^2\right)$$

**(B) 規制ダミー版 (初期: 2004, 2018)** — 3パラメータ:

$$p_1(t) = p_{1,\text{base}} + \beta_{2004} I(t \geq 2004) + \beta_{2018} I(t \geq 2018)$$

**(C) 規制・震災ダミー版 (最終: 2006, 2011, 2018)** — 4パラメータ:

$$p_1(t) = p_{1,\text{base}} + \beta_{2006} I(t \geq 2006) + \beta_{2011} I(t \geq 2011) + \beta_{2018} I(t \geq 2018)$$

遊技業界データブック2025 に基づく訂正:
- 2004 は台数ピーク年（結果）、規制年ではない → **2006（5号機ショック）** に訂正
- **2011（東日本大震災）** を追加

### 2.5 時変 NBD パラメータ

参照年 $t_{\text{2013}} = 2013$ を中心とする対数線形:

$$\log r(t) = r_0 + r_1 (t - t_{\text{2013}})$$

$$\log \alpha_{\text{nbd}}(t) = a_0 + a_1 (t - t_{\text{2013}})$$

### 2.6 二重尤度

**観測1: 参加人口（1994-2019、年次、N=26）**

$$A_{\text{obs}}(t) \sim \mathcal{N}\!\left(M \cdot a(t) \cdot \bigl[1 - (\tfrac{\alpha_{\text{nbd}}(t)}{\alpha_{\text{nbd}}(t) + 1})^{r(t)}\bigr], \; \sigma^2\right)$$

**観測2: 頻度分布（n=1500、9 年、N=10）**

$$\mathbf{n}_t \sim \text{Multinomial}\!\bigl(N_{\text{survey}} = 1500, \; \hat{\mathbf{p}}_{\text{bin}}(r(t), \alpha_{\text{nbd}}(t))\bigr)$$

### 2.7 ビン定義（時間で異なる）

| ビン | 2008-2012 | 2013- |
|---|---|---|
| heavy | $X \geq 104$ | $X \geq 104$ |
| middle | $48 \leq X < 104$ | $48 \leq X < 104$ |
| lmiddle | $12 \leq X < 48$ | $12 \leq X < 48$ |
| light | $5 \leq X < 12$ | $4 \leq X < 12$ |
| rare | $2 \leq X < 5$ | $1 \leq X < 4$ |

## 5. 構造的発見（全モデルで頑健）

### 5.1 NBD 時変の検出

1. **$r$ は不変**: 頻度分布の形（heterogeneity）は 2008-2019 で変化なし
2. **$\alpha_{\text{nbd}}$ は有意に減少**（年率 $-2.2\%$ log units）
3. **$\mathbb{E}[\lambda] = r / \alpha$ は増加**: 2008年 ~59 回/年 → 2019年 ~74 回/年

数値例:
$$\alpha(2008) = e^{-5.17 - 0.022 \times (-5)} \approx 0.0063, \quad \alpha(2019) \approx 0.0050$$
$$\mathbb{E}[\lambda](2008) \approx 58.6, \quad \mathbb{E}[\lambda](2019) \approx 73.8$$

**解釈**: 「**薄い層が抜けて、残った人は heavy/middle 中心の濃いプレイヤー**」という市場構造変化。

### 5.2 スマホ期 $c_2 \approx 2014$ のロバスト性

構造変化点の中心時刻は全モデルで一致:

| モデル | $c_2$ |
|---|---|
| 従来 q2=0 版 | 2014 |
| NBD 単独版 | 2015 |
| Joint (Gaussian) | 2014.03 |
| Regulation (2006,2011,2018) | 2013.26 |

**スマホ普及期を構造変化点として完全に検出**。研究上の確信を持てる発見。

### 5.3 閾値補正の定量

$$P(X = 0 \mid r(2013), \alpha(2013)) = (0.00566 / 1.00566)^{0.369} \approx 0.148$$
$$\therefore P(X \geq 1) \approx 0.852$$

つまり Bass のアクティブ状態 $a(t)$ の人々のうち、年1回閾値を越えるのは約 **85%**。観測参加人口は真の Bass アクティブの約 15% を過小評価。

## 6. 識別性改善の定量評価

| パラメータ | NBD単独 | q2=0 | **Joint** | **Regulation** |
|---|---|---|---|---|
| $p_{2,\text{base}}$ | 1.27 △ | 1.34 △ | **0.69 ○** | 1.33 △ |
| $A_2$ | 3.33 ✗ | 2.40 ✗ | **1.15 △** | 2.29 ✗ |
| $\sigma$ | 1.64 △ | 0.69 ○ | **0.35 ○** | 0.66 ○ |
| $q_1$ | 1.73 △ | ~2.3 △ | 1.46 △ | 2.50 ✗ |

`CI/med` $= (q_{0.975} - q_{0.025}) / |q_{0.5}|$、$< 1$ で○、$< 2$ で△、 $\geq 2$ で✗

### 観察

- **Joint (Gaussian $p_1$) が最も識別性良好** → $p_1$ の滑らかな関数形がデータに合う
- **規制ダミー化は識別性をむしろ悪化** → step 関数が不自然な不連続を強制、他パラに吸収される
- ただし $M, c_2, \alpha_{\text{nbd}}$ 系はどのモデルでも堅牢

## 7. 規制ダミー化の挑戦と教訓

### 7.1 なぜ規制効果は検出できなかったか

1. **年次データ N=26 の情報量不足**: ステップ変化の検出には前後各数年の鮮明な対比が必要
2. **$p_{1,\text{base}}$ への吸収**: 規制は平均値にならされ、ベースラインが規制後の"平均的"値を捉える
3. **データの内生性**: 規制が「プレイヤー減少の原因」か「業界縮小の結果」か判然とせず
4. **規制の多面性**: 2018 規制は射幸性抑制、離脱側 $p_2$ に効く可能性もあり
5. **$p_1$ の弱識別性**: 内因伝播 $q_1 \cdot a$ と外部 $p_1$ の加法形は構造的に分離困難

### 7.2 震災効果の弱い兆候

$\beta_{2011} = -0.037$、95%CI $[-0.24, 0.17]$。**事後の 64% が負の領域** にある。統計的有意性には届かないが、方向性としては「震災による外部加入率の低下」と整合。

震災は 2011年3月で 1年分の観測しか影響できないことも、有意検出の困難さの一因。

### 7.3 得られた方法論的教訓

- **関数形（Gaussian vs step）は識別性の根本問題を解かない**
- **外生イベントは適切な事前信念で使うべき**: 本件ではゼロ中心 Uniform 事前だったため、事後もゼロ近傍に収束
- **情報的事前**（例: 震災効果を負の範囲に限定）で再評価する余地はあるが、恣意性との兼ね合い

## 8. 結論

### 8.1 成果

1. **NBD-Bass 統合ベイズモデルは成立**: 16 パラメータ同時推定で rhat ≤ 1.006 の収束達成
2. **閾値効果の定量化**: $P(X \geq 1) \approx 0.85$ → 観測参加人口は真のアクティブの 85%
3. **$\alpha_{\text{nbd}}$ 有意減少の検出**: 頻度情報が市場の構造変化を独立に裏付け
4. **スマホ期 $c_2 \approx 2014$ の堅牢性**: 全4モデルで一致する構造変化点
5. **識別性の定量改善**: Joint 版で $p_2$ 系パラメータの CI/med が大幅短縮

### 8.2 限界

1. **$p_1/q_1$ の分離困難**: 関数形変更（Gaussian → step）では解決不可
2. **規制イベントの非検出**: 2006/2011/2018 すべて統計的に非有意
3. **N=26 の情報量壁**: 年次データでは細かい時点検出に限界

### 8.3 一般的枠組みとしての意義

本手法は、**Bass 系モデルへ NBD 観測補正を組み込む一般的フレームワーク**として、以下の領域に拡張可能:

- フィットネスサブスク（月1回以上利用）
- SNS 利用（週1回以上アクティブ）
- 遊戯・娯楽習慣全般

任意の閾値 $\tau$ について:

$$A_{\text{obs}}(t) = M(t) \cdot a(t) \cdot \mathbb{P}\bigl(X \geq \tau \mid r(t), \alpha(t)\bigr)$$

潜在状態（Bass 動態）と観測層（NBD 閾値超過）を分離推定できる。

### 8.4 今後の研究方向

- **月次・四半期データ**があれば規制効果の検出力が大幅向上
- **売上・設置台数**との二重尤度で $p_1, q_1$ を構造的に識別
- **変異株別 q₁(t)** のような分野特化拡張
- **4状態拡張**（SEIR型）で潜伏期間を加える
